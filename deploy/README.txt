Instructions for running the LunarX Game
=========================================

In Mac OS X, double click on the file "run.command" in the "deploy" folder.
In Microsoft Windows, double click on the file "run.bat" in the deploy folder.

Requirements:
You will need to have JDK installed

If permission is decline on Mac do the following:

Open the Terminal in the /Applications/Utilities/ folder and type in:
 
chmod u+x

followed by a space. Drag the run.command into the open Terminal window and press Enter.
